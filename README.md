# Exerc-cios-AP2---Java---Algoritmo-e-Estrutura-de-Dados
Projeto Java com implementações de Lista, Pilha e Fila, incluindo menu interativo

- `estruturas/` → contém as classes que implementam os TADs: Lista, Pilha e Fila.  
- `exercicios/` → contém os códigos de teste de cada exercício.

---

## Descrição dos Exercícios

### Exercicio1 – Lista
- Implementa uma lista de inteiros usando vetor.  
- Permite **inserir** elementos e **mostrar** o estado atual da lista.

### Exercicio2 – Pilha
- Implementa uma pilha de inteiros (LIFO).  
- Permite **empilhar**, **desempilhar** e **mostrar** os elementos.

### Exercicio3 – Fila
- Implementa uma fila de inteiros (FIFO) usando vetor.  
- Permite **enfileirar**, **desenfileirar** e **mostrar** os elementos.

### Exercicio4 – Mostrar Estado Atual
- Cria um arquivo que **mostra o estado atual de todas as estruturas**, incluindo ações como empilhar, desempilhar, enfileirar e desenfileirar.

### Exercicio5 – Menu Interativo
- Cria um menu que permite ao usuário **interagir com Lista, Pilha e Fila**, realizando operações e visualizando os estados atuais.

---

## Como Executar

1. Abra o projeto no **IntelliJ IDEA**.  
2. Execute os arquivos dentro da pasta `exercicios/` para testar cada exercício.  
3. Para o **Exercicio5**, o menu interativo permite testar todas as estruturas em tempo de execução.

---
